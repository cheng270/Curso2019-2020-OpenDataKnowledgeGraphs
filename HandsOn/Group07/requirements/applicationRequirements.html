Description of the application

It's a 3D model of Madrid (we plan to use the three.js library to generate it) where you can put the mouse over each zone and it shows you "air quality" data about that zone. You can also visualize the MIN and MAX values by postal code by clicking in the bottom bar.

This is our "design goal" but we could modify this a little bit during the course if we find it to difficult to reproduce.
