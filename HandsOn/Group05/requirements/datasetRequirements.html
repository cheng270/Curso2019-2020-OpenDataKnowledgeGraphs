About the data set:

London Water Quality Data Set satisfy the following requirements:

  • R1. Data set is about Water quality.
  • R2. Data set is available as a CSV file.
  • R3. Data set is provided as open data under the Open Government Licence with no requirement for registration.

    If you make use of this data please acknowledge this with the following attribution statement:

    uses Environment Agency water quality data from the Water Quality Archive (Beta)
  • R4. Data can be easily linked with generic real-world entities (e.g., locations using geographic Cartesian coordinates).
  • R5. Documentation is available for the data set, click on documentaion at the end of the page.
  • R6. Data can come from multiple data sources, not only csv, for example we can use the APIs available, see below documentation for details.
