Dataset requirements

Requirements satisfied

  • R1. Data is in the smart city domain.

    The dataset shows the wi-fi spot distribution in New York city, which is one of the available domains to be covered.

  • R2. Data is available as a CSV file.

    Every file in the dataset has a CSV format.

  • R3. Data has an open license so it can be published.

    The dataset has an open license, as it's stated below.

  • R4. Data can be easily linked with generic real-world entities (e.g. locations).

    The dataset has many entities that can be linked, as it's stated below.

License

All the data present in this dataset comes from "New York City Open Data" website (link).
This dataset is also available in Kaggle (link) , where the type of license is stated (CC-0: Public Domain).
The data is regulated according to the "New York City Open Data Terms of Use" (link) , which allow for free commercial and non-commercial use, and provide no liability on the data.

Entities that can be linked with the dataset

  • Location (Latitude and Longitude)


    Latitude and Longitude data is known for each one of the wi-fi spots.

  • Location (Street Name and Number)


    Street Name and Number data is known for each one of the wi-fi spots and can be linked to their location data (latitude and longitude) and be shown on a map.

  • City


    Neighborhoods in New York

  • Provider (Carrier ID)


    Carrier ID data of each wi-fi spot can be linked to their location data (latitude and longitude) so wifi spot's locations can be checked for each provider.

Entities that can be linked with other datasets

  • Location (Latitude and Longitude)


    Latitude and Longitude data is known for each one of the wi-fi spots.
    Other datasets which include location data can be linked to the location data of our current dataset. For example, this dataset (link), consists in a database of points of interest of the city of New York, based on their latitude and longitude. Therefore, nearby points of interest could be tracked for each wi-fi spot based on its location.
