Defining the resource naming strategy:

 

We are going to use the slash for resources pattern path because of the amount of data generated by sensors and different captures and the hash # for ontological terms paths.

 

The domain of the generated Dataset is:

 http://www.smart-city.GP04-LDATA.org/

 

Ontological terms path:

·        http://www.smart-city.GP04-LDATA.org/city/ontology/Hospital#

Ontological terms pattern:

·        http://www.smart-city.GP04-LDATA.org/city/ontology/Hospital#<term_name>

Example:

·        http://www.smart-city.GP04-LDATA.org/city/ontology/Hospital#Hospital

·        http://www.smart-city.GP04-LDATA.org/city/ontology/Hospital#location

 

Individuals path:

o   http://www.smart-city.GP04-LDATA.org/city/ontology/Hospital#

 

Individuals pattern:

o   http://www.smart-city.GP04-LDATA.org/city/ontology/Hospital#<resource_type>/<resource_name>

Example:

·            http://www.smart-city.GP04-LDATA.org/city/ontology/Hospital#hashospital_Ownership

 

·         http://www.smart-city.GP04-LDATA.org/city/ontology/Hospital#hashospital_Type

 

·         http://www.smart-city.GP04-LDATA.org/city/ontology/Hospital#hasname
