List of Requirements

The four .csv datasets come from the same publisher (Ayuntamiento de Madrid) and match the following requirements:

R1. Data are in one of the selected smart city domains: The data come from the open data portal of the city of Madrid.
R2. Data are available as a CSV file: The datasets have been downloaded in the .csv format.
R3. Data have an open license so they can be published: The license is open and allows for publication. The license is analyzed in the following section.
R4. Data can be easily linked with generic real-world entities: For example, real buildings or city districts.
R5. Documentation exists for the data (optional): The documentation can be found in the following link.
R6. Data can come from multiple data sources (optional).
