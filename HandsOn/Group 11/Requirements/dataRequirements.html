Air Quality in Madrid Data Set satisfy the following requirements:

  • R1-Description.

    Collect the concentration of particles to know if they are harmful to health in the different measurement stations, located in different points of the city of Madrid

  • R2. Data set is available as a CSV file.
  • R3-License.

    All the data present in this dataset comes from Madrid's City Council Open Data website, which are the ones to be acknowledged for the data collection. The data therefore inherits the Madrid Open Data Terms of Use, which allow for free commercial and non-commercial use, and provide no liability on the data.

  • R4. Data can be easily linked with generic real-world entities (e.g., locations, like for example locations as Madrid and the diferents streets where the starion are located).
  • R5-Documentation.

    All the data is extracted from the original files and processed to result in a more convenient format for typical Kaggle purposes. While the original data includes hours as different columns and measurements as different rows, this version is structured the other way round: Each row is timestamped and the columns are the different measures performed at that point in time in a certain stations. This allows faster preparation for time series analysis and prediction tasks.
